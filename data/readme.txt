three 3dm files (rhino 4.0 format) are included as the test data,
they can be read by the java program directly.


their contents:

177 polygon.3dm		177 polygons from the landscape artifact
char.3dm		48 english letters in different size (polygon with holes)
mask.3dm		one polygon with hole, as the irregular packing container


in addition, the 177 polygons are provided in two different format: dxf and txt.
for the txt file, each line of it corresponds to one polygon, using the following format:

x1 y1 x2 y2 ...

dxf file are for AutoCAD and other softwares
